# VidaPlena SpA - Landing Page

Landing page profesional desarrollada para VidaPlena SpA, empresa de servicios 
de bienestar y salud integral. Proyecto desarrollado como certamen práctico 
de Desarrollo Web Front-End en INACAP.

---

## Tecnologías Utilizadas

- HTML5 semántico
- CSS3 (variables, Flexbox, Grid, media queries)
- Google Fonts (Poppins)
- Git y GitHub para control de versiones

## Secciones del Sitio

- **Header**: navegación fija con links a cada sección
- **Hero**: imagen de fondo, titular principal y llamada a la acción
- **Beneficios**: tres servicios presentados en tarjetas
- **Testimonios**: dos opiniones de clientes
- **Contacto**: formulario con nombre, email y mensaje
- **Footer**: información de contacto, redes sociales y copyright

## Uso de IA

Durante el desarrollo de este proyecto utilicé Claude (Anthropic) como 
herramienta de apoyo bajo las siguientes condiciones:

### Prompts utilizados

- *"Explícame qué etiquetas semánticas de HTML5 son las más adecuadas 
para estructurar una landing page"*
- *"¿Cuál es la diferencia entre usar article y section en HTML5?"*
- *"Sugiere una paleta de colores verdes accesible para una empresa 
de bienestar y salud"*
- *"¿Cómo implemento variables CSS para mantener consistencia 
en los colores de mi proyecto?"*

### Cómo adapté los resultados

- La paleta de colores fue sugerida por IA pero ajusté los valores 
hexadecimales para mejorar el contraste y cumplir WCAG AA
- Las explicaciones sobre etiquetas semánticas me ayudaron a decidir 
usar article para tarjetas y testimonios en lugar de div
- El contenido textual de servicios y testimonios fue adaptado 
al contexto específico de VidaPlena SpA
- Todos los fragmentos de código fueron revisados, comprendidos 
y modificados antes de incluirlos en el proyecto
